## Resources

### Cards

A `card` is a Jira card.

``` sh
zen card
```

`list`: List all assigned cards

``` sh
zen card list
```

### Workflows

A `flow` is a workflow.

``` sh
zen flow
```

`apply`: Apply local changes to remote

``` sh
zen flow apply
```

### Pull Requests

A `pr` is a GitHub pull request.

``` sh
zen pr
```

`list`: List all assigned PRs

``` sh
zen pr list
```
